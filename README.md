# Email Extractor

## CodeAlpha Python Programming Internship - Task 3

### 📌 Project Description

This project is a simple Python automation script that extracts email addresses from a text file using Regular Expressions (Regex). The extracted email addresses are automatically saved to a separate text file, demonstrating basic automation, file handling, and pattern matching in Python.

---

## 🚀 Features

- Reads data from a text file
- Extracts valid email addresses using Regular Expressions
- Removes duplicate email addresses
- Saves extracted emails to a new text file
- Displays the total number of emails found
- User-friendly console output

---

## 🛠️ Technologies Used

- Python 3
- Regular Expressions (`re` module)

---

## 📚 Concepts Used

- Regular Expressions (Regex)
- File Handling
- Input/Output
- Lists
- Exception Handling

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/Email-Extractor.git
```

2. Open the project folder.

3. Create an `input.txt` file and add text containing email addresses.

4. Run the program:

```bash
python python_script.py
```

or

```bash
py python_script.py
```
